# Diyet-Kalori Hesaplama Uygulaması
Kullanıcılara uyguladıkları diyetlerinde yardımcı olabilecek, öğün bazında tükettikleri besinlerin girişini yaptırarak günlük alınan kalori ve besin değerlerini hesaplayıp gösteren, kullanıcının profil durumuna göre günlük alması gerektiği kalori miktarını gösteren,kullanıcıya ait günlük-haftalık-aylık detaylı analiz ve raporların bulunduğu masaüstü otomasyon olarak hazırlanmış bir form uygulamasıdır.
